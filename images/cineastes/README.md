# Photos des Cinéastes

Placez ici les photos des cinéastes au format **3:4 (portrait)**.

## Nomenclature

Le nom du fichier doit être le nom du cinéaste normalisé :
- Minuscules uniquement
- Espaces → tirets `-`
- Accents supprimés
- Extension `.jpg`

### Exemples :
- `eric-rohmer.jpg`
- `jim-jarmusch.jpg`
- `mia-hansen-love.jpg`
- `claire-denis.jpg`

## Spécifications

- **Ratio** : 3:4 (portrait) - ex: 400x533px
- **Format** : JPG
- **Poids** : max 200KB
- **Style** : Photo portrait, fond neutre
